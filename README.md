# 🖼️ 鹰岚图片拼合工作台 (Image Stitching Workbench)

这是一个追求极致体验的**纯前端轻量级图片排版与拼合工具**。无需后端服务器支持，不涉及任何图片上传隐私风险，完全基于浏览器原生 Canvas 渲染引擎构建。

## ✨ 核心特性

* **🚀 零依赖，开箱即用**：无需安装任何环境，单 `index.html` 文件即可运行。
* **🖱️ 物理直觉交互**：拖拽图片瞬间互换位置，滚轮实现精准缩放。
* **✍️ 零重影原位编辑 (Seamless Editing)**：双击画板中的悬浮胶囊标签，就地化为输入框。打字即渲染，告别繁杂的弹窗表单。
* **🏷️ 现代化悬浮胶囊排版**：图片 100% 撑满网格，文字与角标采用深色半透明胶囊悬浮于画面之上，最大化视觉张力。
* **📥 纯净模式导出**：支持一键剥离辅助文字，生成高精度的纯净排版图像。

## 🛠️ 如何使用

1. **导入素材**：点击上传本地图片，或直接在页面中 `Ctrl+V` / `Cmd+V` 粘贴截图。支持网络 URL 跨域提取。
2. **重命名与排序**：在左侧列表可直接拖拽调整全局顺序；双击列表名称即可快速改名。
3. **精细排版**：在右侧画板区，按住图片拖动进行位置互换，使用鼠标滚轮调整图片大小。
4. **一键导出**：调整至满意效果后，点击“保存文件”或“复制图片”即可输出高清合并图。
